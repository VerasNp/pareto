## Rodando a aplicação

Primeiramente clone o projeto:

```
git clone https://github.com/VerasNp/pareto.git
```

Instale todas as dependências:

```
npm install
```

Rode a aplicação:

```
npm run dev
```

A aplicação deve abrir em seu navegador em http://localhost:5173/
